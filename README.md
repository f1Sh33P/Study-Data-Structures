## About

1. **部分修改**：在视频代码的基础上，优化了变量名称、代码结构。
   - 能用缩写的，一律缩写。如 vertex 缩写为 vex，adjacency 缩写为 adj。
   - 结构体的命名，使用首字母大写的驼峰法。如 TreeNode，BiTree。
   - 函数的命名，使用普通的驼峰法。如 initQueue()，createTree()。
   - 变量的命名，使用下划线。如 vex_num，tree_arr。
2. **AI 辅助**：使用 Gemini 优化了部分代码逻辑。
3. **心得笔记**：补充了许多注释，记录了笨人学习过程中遇到的困难和应对思路。
4. **可视化展示**：树和图的代码，凡是有实例的，都使用 ASCII 字符及 Unicode 的 Box Drawing 中的字符进行了可视化展示。
